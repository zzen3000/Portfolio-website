## 📖 What is this?
This is a custom, fully responsive portfolio website I built for my friend. 

Full transparency: This isn't a paid gig or a professional template for commercial use. It was an individual assignment for my Web App Development course as part of my Multimedia Computing degree at UNIMAS. 

## ✨ Features (The Cool Stuff)
Even though it's a university assignment, I made sure the UI/UX actually slaps. Here's what's under the hood:

* Dark/Light Mode Toggle 🌗: Built-in theme switcher that actually remembers your vibe using `localStorage`. (Defaults to dark mode because we have rights).
* Responsive Sidebar 📱: A sleek, mobile-first navigation menu with a hamburger toggle that slides perfectly on smaller screens.
* Scroll Spy Navigation 🧭: The sidebar links actively highlight based on whatever section you are currently doomscrolling through.
* Interactive Contact Form ✉️: It doesn't actually hook up to a backend (yet), but it fakes an API call with a "Sending..." state and a success message so the UX feels 100% real.
* A11y Friendly ♿: Includes screen-reader-only (`sr-only`) classes and custom focus states so keyboard navigation is actually usable.

## 🛠️ Tech Stack
Kept it strictly vanilla. No frameworks, no bloated libraries. Just raw code.
* HTML5 (Semantic structuring)
* CSS3 (Custom variables, Grid/Flexbox layouts, CSS animations)
* Vanilla JavaScript (DOM manipulation, event listeners, state handling)
